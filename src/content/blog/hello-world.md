## The publishing pipeline

Fun detail: posts here are just markdown files in a Git repo. I draft them in a
conversation with Claude, and when I give the green flag, the post gets
committed straight to GitHub — Vercel picks it up and it's live about a minute
later. No CMS, no admin panel, no friction.
